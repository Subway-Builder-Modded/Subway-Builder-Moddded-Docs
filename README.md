# Subway-Builder-Modded-Docs

The official modded Subway Builder documentation (https://subwaybuildermodded.com)

## Usage

The following are useful commands to run before contributing:

Install dependencies

```bash
pnpm install
```

Run locally in development mode

```bash
pnpm run start
```

Build static docs

```bash
pnpm run build
```

Serve the production build locally

```bash
pnpm run serve
```

Lint and auto-fix supported files

```bash
pnpm run lint
```

Check .md / .mdx format with prettier

```bash
pnpm run format:check
pnpm run format
```

:warning: Please make sure that there are spaces around the admonitions start and end directives
(see Docusaurus
[docs](https://docusaurus.io/docs/markdown-features/admonitions#usage-with-prettier))

Deploy and theme customization

```bash
pnpm run deploy
pnpm run swizzle
```
