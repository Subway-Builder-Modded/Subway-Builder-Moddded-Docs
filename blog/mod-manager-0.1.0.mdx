You can download the latest release of the Subway Builder Mod Manager at: https://github.com/gio1112/subway-builder-mod-manager/releases/tag/v0.1.0. Update your Mod Manager before proceeding with any guide.

### Changelog:

#### Core Features:

 - Intelligent Mod Browser - Browse all available mods from the official repository with real-time search and filtering
 - One-Click Installation - Seamless mod installation directly to your game directory
 - Update Management - Automatic detection and notification when mod updates are available (To be tested)
 - Favorites System - Bookmark and organize your preferred mods for quick access
 - Theme Customization - Toggle between light and dark modes with persistent preferences
 - Live Statistics - Dashboard tracking total mods, installations, and favorites
 - Advanced Search - Filter by name, author, description, or keywords

#### Technical Highlights:

 - Standalone executable - no Python or external dependencies required
 - Lightweight footprint with optimized performance
 - Local storage for user preferences and mod metadata
 - GitHub-integrated mod discovery and versioning
